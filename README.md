# 🛍️ Ứng Dụng Mua Sắm Quần Áo Tích Hợp AI

Dự án xây dựng một nền tảng thương mại điện tử chuyên cung cấp các mặt hàng thời trang, được tích hợp trí tuệ nhân tạo (AI) nhằm tối ưu hóa và nâng cao trải nghiệm mua sắm của người dùng.

---

## 🛠️ Công Nghệ Sử Dụng

*   **Backend:** C#, Framework ASP.NET Core MVC
*   **Frontend:** HTML, JavaScript, Tailwind CSS
*   **Cơ sở dữ liệu:** SQL Server
*   **Công cụ thiết kế & Quản lý:** 
    *   Quản lý mã nguồn: GitHub
    *   Thiết kế hệ thống: Draw.io, Power Designer
    *   Phân tích dữ liệu: Power BI

---
